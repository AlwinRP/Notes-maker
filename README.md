# Notes-maker
 MERN stack webapp
 ## Server-Setup ##
 To run the app run the server script from the server folder

```javascript
   cd server

   npm start
```
```
> server@1.0.0 start
> nodemon index.js

[nodemon] 3.1.3
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,cjs,json
[nodemon] starting `node index.js`
Server is Running
```
